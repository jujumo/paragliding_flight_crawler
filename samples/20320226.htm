Le vol de JACQUES FOURNIER du 28/04/2022

  • date : 28/04/2022
  • type de vol : triangle
  • département : 38
  • décollage : Saint Hilaire déco sud
  • atterrissage : Lumbin
  • distance totale : 168.89 km
    • b1-b2 : 82.48 km
    • b2-b3 : 59.94 km
    • b3-b1 : 26.36 km
  • points : 202.67 pts
  • durée (du parcours) : 6h54mn
  • vitesse (sur le parcours) : 24.5km/h
  • Valide et pris en compte dans les classements  
  • Aile : K Ozone - Enzo 3 (CIVL Competition Class)

    Club : PARATEAM -

  • balises

    • bd : Saint hilaire du touvet
    • b1 : Rachais
    • b2 : Dents de Lanfon
    • b3 : St Genix
